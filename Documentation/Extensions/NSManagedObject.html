  • nextId(inContext:withCompletion:)

    It returns you the smallest free id not used by any other object from this NSManagedObject subclass. NOTE: your NSManagedObjects has to store their ids in attribute name called ‘id’.

    How it works:

    • it fetches the object with the biggest id currently stored on in CoreData
    • it returns this biggest id + 1

    • if there are no records for this Entity in CoreData then it returns 1.
